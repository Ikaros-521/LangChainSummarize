# langchain_gpt_bot

## 环境
系统：win10  
python：3.10.8  
依赖：`pip install python-dotenv langchain gpt_index==0.5.27`  

## 配置
`.env`中配置你的`OPENAI_API_KEY`
```
OPENAI_API_KEY=<your valid openai api key>
```

## 运行
`python langchain_gpt_bot.py docs`
